Edit Items - Tips
What is the Item Module?

Raptech allows users to manage all items that are purchased, manufactured, sold, or kept in inventory. Services can also be defined as items, but they are only relevant for sales. Enter the data for each item that will be used for Purchasing, Sales, Production, Warehousing, Accounting, and Services. The Item tab is where you add, update, search for, and manage your item data. This item tab contains a list of all the items in the company.

Pre-requisites: Some prerequisites are required as a prior condition and should be added by Admin Users under admin settings, such as UOM, Category, Ledger accounts of Purchase and Sales and Additional fields (if required).

What steps do I take to edit an item?
To successfully edit an Item, follow these steps:

  1. Items sub-module should be chosen under the Item module.
    In search mode, the Items tab appears. Switch to the edit mode.
  2. Fill in the necessary info under each of the following tabs:
    (Fields marked with an asterisk (*) are required, and mandatory.)
    1. Primary Details

      On the edit page in this section, none of the fields can be edited.

    2. Item Details
      • If the Item No. is a manual entry, enter it here.
      • Enter the Item Description.
      • If necessary, enter an Additional Description of the item.
      • Provide the item's UOM.
      • If the item has multiple UOMs, uncheck the alternate UOM check box and enter the alternate UOM details.
      • If available, enter the Manufacturer and Manufacturer Part No.
      • If required, select the Category in which the item falls.
      • Upload the item's image.
    3. Purchase Related Details
      • Enter the Minimum Order Qty value to receive an alert message when the item value is reached in the inventory.
      • When the item reaches the minimum level, select the Criticality type required.
      • Select the GRN Inspection Required check box if the item needs to be inspected while being received from vendors.
      • If required, select the category under which the item belongs.
      • Enter the Barcode details.
      • If required, enter the item's expiry days.
      • If necessary, select the catch weight check box and enter the item's nominal weight.
      • Select the item issue method from the drop-down list.
    4. Finance Details
      • Enter the item's HSN/SAC code.
      • Enter the CESS details if required.
      • Select Purchase and Sales Accounts codes to which the item should be linked.
    5. Depreciation Details
      • For the Asset items, enter the depreciation details such as Depreciation Method, Depreciation Based on, Useful Life, Salvage Value, Depreciation Percentage, Depreciation Frequency, Depreciation Expense Account, and Accumulated Depreciation Account.
    6. Scrap Item
      • If it is a scrap item, enter the scrap value.
  3. After you've filled out all of the details, click Create.
